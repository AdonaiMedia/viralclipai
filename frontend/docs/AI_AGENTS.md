# ViralClip AI Agents

## Overview

ViralClip AI uses specialized AI agents.

Each agent has one responsibility.

The Mission Planner coordinates all agents.

---

# Analysis Agent

Purpose

Analyze uploaded videos.

Responsibilities

- Speech analysis
- Emotion detection
- Hook detection
- Engagement prediction

---

# Clip Agent

Purpose

Generate viral clips.

Responsibilities

- Find viral moments
- Cut clips
- Rank clips

---
